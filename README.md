

# YouTube Comment Sentiment & Emotion Analyzer

A high-performance, full-stack application that goes beyond simple sentiment analysis. It performs deep psychological profiling of YouTube comments using a custom, fine-tuned multi-label emotion model (28 categories) and provides actionable insights for content creators.

## 📋 Overview

This project fetches YouTube comments and processes them through an ensemble of AI models to identify not just *how* a viewer feels (positive/negative), but *what* specific emotions (joy, anger, disappointment, curiosity, etc.) they are expressing.

## ✨ New Features (2026 Update)

* 🧠 **Multi-Label Emotion Model**: Upgraded from a 7-label model to a comprehensive **28-emotion GoEmotions-based model**.
* 🚦 **Sigmoid Threshold Inference**: Allows for multiple emotional tags per comment (e.g., a single comment can be "gratitude" AND "joy").
* 💡 **AI-Powered Suggestions**: Automatic, context-aware advice generated by analyzing dominant emotion trends.
* 📉 **Advanced UI**: Updated React dashboard with 28-color emotion mapping and "show all/top 10" toggle.
* ⚖️ **Mixed Sentiment Detection**: Automatically flags comments that express conflicting opinions (e.g., "Great video, but audio is bad").

## 🛠️ Tech Stack

**Backend:**

* Python 3.x | FastAPI | PyTorch | Hugging Face Transformers
* **Inference Engine**: Custom `DistilRoBERTa` fine-tuned for multi-label classification.
* **MLOps**: MLflow for experiment tracking and run logging.

**Frontend:**

* React 19.2 | Vite | Recharts | Axios

## 📦 Setup & Installation

### 1. Environment Setup

```bash

git clone <your-repo-url>
cd sentiment-analyzer

# Create and activate environment
python -m venv venv
venv\Scripts\activate  # Windows
source venv/bin/activate  # Linux/macOS

# Install dependencies
pip install fastapi uvicorn transformers torch pandas google-api-python-client python-dotenv mlflow

```

Create a `.env` file in the root directory:

```env
YOUTUBE_API_KEY=your_youtube_api_key_here

```

### 3. Run the Application

**Terminal 1 (Backend API):**

```bash
uvicorn api:app --reload

```

**Terminal 2 (Frontend Dashboard):**

```bash
cd dashboard
npm install
npm run dev

```

## 🧠 Model Logic

The application uses a **Multi-Label Classification** strategy.

* **Thresholding**: A probability threshold of `0.3` is applied to model logits. If an emotion score exceeds this, it is flagged as "present."
* **Fallback**: If no emotions cross the threshold, the model defaults to the highest-scoring label to ensure no data is lost.

## 📊 Dashboard Features

* **Sentiment Split**: Real-time Pie chart of Positive/Neutral/Negative/Mixed feedback.
* **Emotion Breakdown**: Interactive bar chart displaying all 28 detected emotions.
* **AI Suggestions**: Dynamically generated cards providing data-driven advice (e.g., "High negative sentiment detected" or "Viewers want a part 2").
* **Smart Filtering**: Filter by specific sentiments or search through comment text instantly.
